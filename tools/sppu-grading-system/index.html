Understand SPPU Grading

SPPU follows a 10-point grading system to evaluate student performance. Your mark sheet typically shows only the grade (e.g., A+, B). This tool helps decode those grades into a rough marks range so you can estimate your performance more clearly.

You can explore more academic tools like SGPA to CGPA, CGPA to Percentage, and more on our Academic Tools page.
